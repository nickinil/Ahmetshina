## 1. Проверяемые функции

В ходе тестирования проверялись следующие функции:

- запуск графического интерфейса;
- добавление новой записи;
- проверка корректности ввода;
- сохранение данных в JSON;
- загрузка данных из JSON;
- фильтрация данных;
- удаление или очистка записей;
- корректная работа таблицы или списка.

---

## 2. Позитивные тесты

| № | Действие | Входные данные | Ожидаемый результат | Результат |
|---|---|---|---|---|
| 1 | Запуск программы | `python main.py` | Открывается окно приложения | Успешно |
| 2 | Добавление корректной записи | Заполнены все обязательные поля | Запись появляется в интерфейсе | Успешно |
| 3 | Сохранение данных | Добавить запись или нажать кнопку сохранения | Данные сохраняются в JSON-файл | Успешно |
| 4 | Загрузка данных | Перезапустить программу | Данные из JSON загружаются обратно | Успешно |
| 5 | Фильтрация данных | Указать условие фильтра | Отображаются только подходящие записи | Успешно |
| 6 | Очистка или удаление записи | Выбрать запись и нажать удаление | Запись удаляется из интерфейса и JSON | Успешно |

## 3. Негативные тесты

| № | Действие | Входные данные | Ожидаемый результат | Результат |

## 4. Граничные тесты

| № | Действие | Входные данные | Ожидаемый результат | Результат |
|---|---|---|---|---|
| 1 | Минимально допустимое значение | Минимальное корректное значение | Запись добавляется | Успешно |
| 2 | Большое значение | Большое число или длинный текст | Запись добавляется и сохраняется | Успешно |
| 3 | Пустой JSON-файл | `[]` или пустая структура | Программа запускается без ошибок | Успешно |
| 4 | Русские символы | Данные на русском языке | Данные корректно сохраняются и загружаются | Успешно |

---

## 5. Проверка JSON

Проверено:

- данные сохраняются в JSON;
- данные загружаются после перезапуска;
- русские символы отображаются корректно;
- после удаления или очистки записей JSON обновляется.
