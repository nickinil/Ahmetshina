# 🏋️ Training Planner - План тренировок

**Автор:** Ахметшина Аэлита Дмитриевна

## 📝 Описание проекта

Training Planner - это desktop-приложение для планирования и учёта тренировок с графическим интерфейсом на Python (tkinter). Приложение помогает отслеживать физическую активность, анализировать статистику и сохранять историю тренировок.

### ✨ Основные возможности

- ✅ **Добавление тренировок** с указанием даты, типа и длительности
- ✅ **Просмотр списка** тренировок в удобной таблице
- ✅ **Фильтрация** по типу тренировки и дате
- ✅ **Быстрые фильтры** (сегодня, эта неделя)
- ✅ **Редактирование** существующих записей
- ✅ **Удаление** одной или нескольких тренировок
- ✅ **Статистика** (общее количество, общее время, средняя длительность, самый популярный тип)
- ✅ **Экспорт данных** в JSON файл
- ✅ **Автосохранение** в JSON при каждом изменении
- ✅ **Валидация** ввода (формат даты, положительная длительность)
- ✅ **Контекстное меню** для быстрого копирования данных
- ✅ **Примеры данных** для демонстрации при первом запуске

## 🚀 Установка и запуск

### Требования
- Python 3.7 или выше
- Стандартные библиотеки Python (tkinter, json, datetime, os)

### Запуск
```bash
python training_planner.py
